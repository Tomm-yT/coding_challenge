# Coding Challenge for Walmart
A simple app that consumes api data containing a list of countries, and then displays that data in a recycler view with card views.
# Libraries/Patterns: 
- MVVM and LiveData
- Coroutines and Flows
- Retrofit
- Repository Pattern
